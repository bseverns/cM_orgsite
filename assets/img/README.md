# Placeholder Asset Notes

This folder is ready for real brand assets, student photography, partner logos, and impact visuals.

Current files are abstract placeholders used to keep the layouts stable before final assets exist:

- `hero-network.svg`: default illustration for mission and partnership sections
- `community-grid.svg`: alternate illustration for program and local-context sections

When replacing placeholders:

1. Prefer optimized SVG or compressed WebP/PNG.
2. Keep the same filenames only if you want existing markup to update automatically.
3. Add descriptive alt text in the referencing HTML when the image conveys real information.
4. Avoid decorative stock photos that contradict the local, community-rooted tone of the site.
